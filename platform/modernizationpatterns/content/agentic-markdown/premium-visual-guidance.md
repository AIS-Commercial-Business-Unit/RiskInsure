# Premium Visual Guidance

## Layout principles
- Use fewer, stronger sections.
- Let the headline breathe.
- Prefer clean bands, rails, and grouped cards over cluttered grids.
- Use a restrained accent color to create hierarchy rather than decoration.

## Diagram principles
- Diagrams should look like strategy slides, not app dashboards.
- Use strong titles, short labels, and grouped concepts.
- Put supporting detail underneath, not inside the core visual.
- Make each visual understandable in under 10 seconds.

## Navigation principles
- Add section anchors on long pages.
- Keep overview pages scannable.
- Make drill-down pages feel related through common layout patterns.

## Tone
- Crisp
- Executive
- Trustworthy
- Structured
- Commercially credible
