# Agentic Engineering Use Cases

## Legacy modernization

Use DDD and event storming to rediscover the domain, apply modernization patterns, and introduce governed human-agent delivery across design, build, release, and operations.

## Regulated software delivery

Apply the governance model to define risk tiers, human checkpoints, evidence packs, and policy-based controls while using agents in lower-risk, high-volume engineering work.

## Platform-enabled modernization

Establish an internal developer platform with reusable workflows, context assets, templates, agent guardrails, and runbooks aligned to the operating model.
