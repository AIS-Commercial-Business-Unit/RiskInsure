# Human and Agent Roles

## Why this matters

The model becomes more credible when the site shows who does the work, who stays accountable, and which agents participate at each stage. This turns the site from a conceptual framework into an operating model.

## Human roles

The core human roles are:
- Executive Sponsor
- Business / Product Owner
- Domain Architect
- Platform Engineer
- Security / Compliance Lead
- SRE / Operations Lead
- Enablement Lead

## Agent roles

The core agent roles are:
- Coordinator Agent
- Planning Agent
- Implementation Agent
- Testing Agent
- Security Review Agent
- SRE Agent
- Documentation Agent

## Website recommendation

Create a page that shows:
- role purpose
- where the role participates
- what decisions or responsibilities it owns
- what artifacts it touches
